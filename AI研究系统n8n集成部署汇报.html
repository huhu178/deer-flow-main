🔄 使用流程
团队成员使用流程： 1. 打开AI研究助手界面 (ai-research-interface.html) 2. 选择或输入研究主题 3. 配置语言和研究深度参数 4. 点击"🚀 开始AI研究"按钮 5. 系统自动调用n8n工作流 6. n8n触发AI研究系统生成报告 7. 报告自动保存到outputs/reports/目录 8. 用户收到完成确认和报告信息
技术实现流程： HTML界面 → n8n Webhook → AI Research Service工作流 → HTTP Request → AI系统(/api/chat/stream) → 报告生成 → 文件存储 → 结果返回
⚙️ 部署配置

服务端口配置：

• n8n工作流平台：http://localhost:5678 • AI研究系统API：http://localhost:8000 • 前端界面：file:///path/to/ai-research-interface.html

核心文件清单：

├── ai-research-interface.html # 用户操作界面 ├── n8n-ai-workflow.json # n8n工作流模板 ├── install-n8n.bat # n8n安装脚本 ├── test_api.py # API测试脚本 ├── src/server/enhanced_report_api.py # 增强报告API └── outputs/reports/ # 报告存储目录

n8n工作流配置：

• 工作流名称：AI Research Service • 触发器：When Executed by Another Workflow • HTTP请求：POST localhost:8000/api/chat/stream • 超时设置：300秒 • 参数支持：topic, language, depth
